What is EQLogParser?

EQLogParser is a real-time combat log analyzer and damage parsing application built specifically for the EverQuest MMO. It monitors and processes in-game log files to provide detailed statistics as well as various utility functions

  • Damage dealt and received (per player, mob, or encounter)
  • Spell casting counts and activity timelines
  • Audio triggers and game overlays (damage meter, visual feedback and timers)
  • Log search and automated backups
  • Much more

What do I need to run EQLogParser?

  • Windows 10 or newer (64 bit only)
  • .NET Desktop Runtime 8.0.11+ (download .Net 8 here)
  • Game overlays reqire running Everquest in windowed mode
